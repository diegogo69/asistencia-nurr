### Cuadro de Escenario

|  |  |
| --- | --- |
| **Nombre:** | Generar Reportes de Asistencia |
| **Actores:** | Coordinador / Administrador, Sistema (Generador de Reportes) |
| **Actividades:** | Consolidación de datos históricos, procesamiento de métricas de asistencia y exportación de informes en formatos PDF y CSV/Excel |
| **Detalles:** | El Coordinador o Administrador genera reportes consolidados y analíticos de las marcaciones de asistencia (utilizados para auditoría interna o para cotejar los Informes de Control de Visitas de los pasantes). El Sistema consulta las transacciones, calcula los acumulados de horas laboradas y el porcentaje de cumplimiento, permitiendo exportar el informe en archivos independientes en formato PDF (para impresión o archivo físico) o CSV/Excel (para análisis de datos). |
| **Eventos:** | **Configurar Parámetros:** Selección del rango de fechas, usuario, departamento y formato de exportación (PDF o CSV/Excel). |
| | **Procesar Consolidado:** Cálculo automático realizado por el Sistema para contabilizar horas trabajadas, puntualidades, retrasos e inasistencias. |
| | **Exportar en PDF:** Generación y descarga de un archivo PDF estructurado con encabezados institucionales, listo para firma e impresión. |
| | **Exportar en CSV / Excel:** Generación y descarga de un archivo en formato plano o hoja de cálculo para procesamiento cuantitativo. |
| **Extensiones:** | **1. Configurar Parámetros del Reporte** 1.1 El actor ingresa a la sección "Generación de Reportes". 1.2 Selecciona el rango de fechas (fecha desde / fecha hasta), pudiendo filtrar por un usuario específico, un departamento o generar un consolidado general. 1.3 Selecciona el formato de exportación deseado (*PDF* o *CSV/Excel*). |
| | **2. Procesar Consolidado (Proceso del Sistema)** 2.1 El Sistema consulta las tablas `asistencias` y `usuarios` según los filtros indicados. 2.2 Recorre las entradas y salidas registradas, calculando las horas efectivas acumuladas por cada usuario en el rango seleccionado. 2.3 Agrupa los totales por categoría de cumplimiento (`puntual`, `retraso`, `anticipada`, `incompleto`). |
| | **3. Exportar en Formato PDF** 3.1 Si el actor selecciona la opción PDF, el Sistema compila la información en un documento estructurado que incluye el membrete de la ULA-NURR, datos del departamento, desglose detallado por usuario, resumen de horas acumuladas y línea de firma para la validación del Coordinador. 3.2 El Sistema abre una vista previa descargable en una nueva pestaña del navegador. |
| | **4. Exportar en Formato CSV / Excel** 4.1 Si el actor elige la opción CSV/Excel, el Sistema estructura las filas de marcación y totales en un archivo delimitado por comas u hoja de cálculo 4.2 Inicia la descarga automática del archivo en el dispositivo del usuario. |
| | **5. Validación de Datos Vacíos** 5.1 Si no existen registros de asistencia en el rango de fechas seleccionado, el Sistema interrumpe la generación y despliega una alerta: "No existen registros disponibles para generar el reporte en el rango especificado". |
