### Cuadro de Escenario

| Componente | Especificación |
| --- | --- |
| **Nombre:** | Registrar Biometría Facial |
| **Actores:** | Coordinador / Administrador, Sistema (Procesador Biométrico)

 |
| **Actividades:** | Captura de imagen facial mediante cámara web, extracción de puntos característicos y generación/almacenamiento del vector biométrico del usuario

 |
| **Detalles:** | El Coordinador o Administrador ejecuta el módulo de enrolamiento biométrico para un usuario. El Sistema activa la cámara web, detecta el rostro del usuario en tiempo real, extrae sus patrones geométricos faciales, genera un hash o representación matemática única (`vector_biometrico`) y lo almacena asociado al perfil del usuario para su posterior autenticación en el marcaje de asistencia.

 |
| **Eventos:** | **Iniciar Captura:** Habilita el dispositivo de vídeo (cámara web) y muestra la vista previa en pantalla.

<br>

<br>**Capturar y Validar Rostro:** Toma la muestra fotográfica y valida condiciones mínimas de iluminación, encuadre y detección del rostro.

<br>

<br>**Generar y Guardar Vector Biométrico:** Extrae la firma matemática y la almacena en el campo `vector_biometrico` del usuario en la base de datos.

<br>

<br>**Reintentar:** Limpia la captura previa y reinicia el flujo de cámara en caso de error o baja calidad.

 |
| **Extensiones:** | **1. Iniciar Captura (Habilitación del Dispositivo)**<br>

<br>1.1 El Administrador o Coordinador presiona la opción "Registrar Biometría Facial" desde el módulo de perfil o gestión de usuario.

<br>

<br>1.2 El Sistema solicita permisos de acceso a la cámara web conectada al equipo.

<br>

<br>1.3 Si la cámara no está disponible o no se otorgan los permisos, el Sistema muestra una alerta: "Dispositivo de cámara no detectado o sin permisos" y cancela la operación.

<br>

<br>

<br>**2. Capturar y Validar Rostro (Proceso del Sistema)**<br>

<br>2.1 El Sistema muestra una guía visual (óvalo de encuadre) en la vista previa de la cámara.

<br>

<br>2.2 El usuario se posiciona frente a la cámara y el Sistema realiza una detección automática de presencia facial.

<br>

<br>2.3 El actor confirma la toma presionando el botón de captura.<br>

<br>2.4 El Sistema evalúa la calidad de la imagen (nivel de luz, enfoque y detección de un único rostro).

<br>

<br>2.5 Si la calidad es insuficiente o no se detecta un rostro claro, el Sistema muestra el mensaje: "Rostro no detectado correctamente o baja iluminación. Por favor, reintente" y habilita la opción Reintentar.

<br>

<br>

<br>**3. Generar y Guardar Vector Biométrico**<br>

<br>3.1 Una vez validada la captura, el algoritmo del Sistema procesa la imagen para transformar las características faciales en un vector de características serializado (`vector_biometrico`).

<br>

<br>3.2 El Sistema guarda la representación matemática resultante en la tabla `usuarios` asociada al usuario correspondiente.

<br>

<br>3.3 Muestra un mensaje de éxito: "Patrón biométrico facial registrado correctamente" y finaliza el proceso.
