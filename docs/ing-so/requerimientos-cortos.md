# Requerimientos Funcionales (RF)

## Módulo 1: Configuración Global y Reglas de Negocio
- **RF-1.1:** Gestionar Unidades Organizacionales.
- **RF-1.2:** Gestionar Horarios.
- **RF-1.3:** Gestionar y validar Rangos de Red Autorizados.
- **RF-1.4:** Autenticar Usuarios Administrativos.

## Módulo 2: Gestión de Usuarios
- **RF-2.1:** Gestionar Usuarios.
- **RF-2.2:** Validar Registro Único (impedir duplicidad).
- **RF-2.3:** Registrar Biometría Facial.
- **RF-2.4:** Generar Código QR de identificación.

## Módulo 3: Autenticación y Registro de Asistencia
- **RF-3.1:** Registrar y validar Asistencias.
- **RF-3.2:** Detectar y resolver Asistencias Incompletas.

## Módulo 4: Auditoría y Reportes
- **RF-4.1:** Consultar Historial de Asistencia.
- **RF-4.2:** Generar Reportes.
